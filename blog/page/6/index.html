忘记mysql密码的处理方式

Mar 7th, 2013

一次又一次的忘记mysql密码，一次又一次的在网上搜索解决方式。

博客真是个好东西，赶紧记录下来吧

废话少说，进入正题

1. 跳过权限检测方式，启动数据库 

linux:
  mysqld --skip-grant-tables
windows:
  mysqld-nt.exe --skip-grant-tables
2. 登陆进去后，修改密码 
mysql -uroot mysql
update user set password = password('forget') where user = 'root';
quit
3. ok，万事大吉
